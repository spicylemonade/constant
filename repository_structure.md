# Repository Structure

This repository contains research aimed at tightening bounds for the Chvátal-Sankoff constant for binary alphabets.

- `research_rubric.json`: Tracks the progress and acceptance criteria for all phases of research.
- `update_rubric.py`: Helper script to automatically update the rubric statuses.
- `results/`: Directory designed to store output data (JSON/CSV) generated by experiments.
- `figures/`: Directory to store generated publication-grade figures (PNG and PDF).
- `sources.bib`: BibTeX file for all sources cited during the literature review.
- `.archivara/`: Contains utility scripts for Semantic Scholar (`semantic_scholar.py`) and Concept Evolve (`concept_evolve.py`).
- `results/concept_evolve/tree/`: (To be created) Will contain exploration nodes for bounds concepts.
