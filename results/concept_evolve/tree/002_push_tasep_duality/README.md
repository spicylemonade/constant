# push_tasep_duality

## Context
Frames the LCS alignment problem as a variant of the Push Totally Asymmetric Simple Exclusion Process (PushTASEP), mapping the growth rate of the LCS to the steady-state particle current.

## Implementation Backlog
- Mathematical setup: \lim_{t \to \infty} \frac{J(t)}{t} = f(\rho) \implies \gamma_2 = \sup_{\rho} f(\rho)
- Try to run the experiment: Simulate frog dynamics (Bukh & Cox 2022) for small N, track particle current J, and derive its exact generating function.
